# MySQL Practice Queries

A collection of MySQL practice queries, database design exercises, and SQL problem-solving tasks to improve database concepts and query-writing skills.

## 📌 About

This repository contains:

- Basic to advanced MySQL queries
- Database schema design exercises
- SQL problem-solving challenges
- Joins, subqueries, aggregate functions, and window functions
- Real-world database practice examples
- Interview-oriented SQL questions

The goal of this repository is to strengthen SQL fundamentals and improve hands-on database experience.

---

## 🛠 Topics Covered

- SELECT statements
- WHERE, ORDER BY, GROUP BY
- JOINS (INNER, LEFT, RIGHT, FULL)
- Aggregate Functions
- Subqueries
- Views
- Stored Procedures
- Triggers
- Indexes
- Window Functions
- Database Normalization
- Constraints & Relationships
